TensorFlow.js: распознаватель цифр со сверточными слоями

Обучите модель распознавать рукописные цифры из базы данных MNIST, используя API tf.layers.

ОПИСАНИЕ

Этот пример позволяет обучить распознаватель рукописных цифр с помощью либо сверточной нейронной сети (также известной как ConvNet или CNN (Convolutional Neural Network)), либо полностью связанной нейронной сети (Connected Neural Network - также известной как DenseNet).

Набор данных MNIST используется в качестве обучающих данных.
